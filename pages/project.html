About Project

Platform aplikasi mobile ReproEd yang dirancang berbasis android ini untuk mempermudah para siswa untuk belajar dan mengetahui lebih dalam terkait Reproduksi dan Edukasi yang didapatkan didalamnya. Perancangan ini menggunakan framework Flutter untuk mempermudah implementasi data dan mempermudah penggunaan aplikasi ReproEd.

Integrasi data dengan BLoC (Business Logic Component) menghubungkan UI, event, dan data API. BLoC bertindak sebagai pengelola logika, menerima event dari UI, memprosesnya, lalu mengirimkan respons API kembali ke UI. Pendekatan ini memastikan status aplikasi dikelola secara konsisten dan terorganisir.
